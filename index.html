Welcome To the HTML5 tutorial

In this website you can learn how to create a HTML5 webpages from beginners level.

There are some steps to be followed in order to create the webpage

  1. Open the notepad file in your computer
  2. Write the Html code in your notepad file
  3. Click on save as
  4. Save your file with the extension of .html
  5. Select all files from the drop down menu
  6. Click enter to save the file
  7. Open the file where it is saved
  8. You can see your first webpage
